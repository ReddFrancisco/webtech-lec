CSS Preprocessors
 * SASS - syntactically awesome style sheets
	- extension
	- most popular
	- developed on top of ruby
 * LESS	- ALMOST SIMILAR TO SASS but it is on javascript
	- another popular preprocessor
 * SCSS - Sassy CSS
	- nested syntax	
	- you can declare variables (ex . $col: #000)
	- you can define a function

** decreasing value of #1 syntax
 example:
 @for $i from 1 through 6 {
   h#($1) {
    font: {
	font: serif;
	size: 30px - ($i-1)*4;
   } 
  }
 }
